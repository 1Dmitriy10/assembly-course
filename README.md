#assembly-course
## **Это учебный сайт про компьютеры и курсы по их сборке.**
## В этом проэкте я отрабатывал навыки html,css,js.
- Подключен swipe слайдер.
- Подключен fontawesome.
- кликабельные элементы имеют анимацию и подсвечиваются.
- Логотипы и таймер обновляются через setInterval.
- На сайте динамический адаптив.
- бургер-меню.
- весь проэкт собран в webpack.
- изображения оптимизированы.
 ## Ссылка на сайт - (https://1dmitriy10.github.io/assembly-course/)
 ## Для запуска проэкта необходимо указать в терминале :
 - npm run dev для сборки в режиме development
 - npm run prod для сборки в режиме production
 - убедиться что ghpages в index.js закомментирована.
 - ## Структура проэкта:
 - dist(собранные файлы сборщиком)
 - src(исходники)
   -  assets(фотографии разбитые по папкам. Названия идентичны блокам в scss)
   -  js(файлы js. Все файлы импортированы в index.js)
   -  style(стили сайта. Главный - style.scss)
     - blocks(для каждой секции в html сделаны свои стили. Они импортированы в style.scss)
 - index.htm
 - package-lock.json
 - package.json
 - webpack.config.js(настройка работы и сборки webpack)
